# Foundation for Speed


### movement + edit


| Shortcut  | Alias                     | Function                                 |
| --------- | ------------------------- | ---------------------------------------- |
| o         | enter line below (INSERT) | enter line below and go into insert mode |
| O         | enter line above (INSERT) | enter line above and go into insert mode |
| (SHIFT) P | paste line boave          | paste yanked / deleted line above        |


| Shortcut |      Alias       |                       Function |
| :------- | :--------------: | -----------------------------: |
| a        | move over INSERT | move over and into INSERT mode |



| Shortcut |                 Alias                  |                                      Function |
| :------- | :------------------------------------: | --------------------------------------------: |
| I(SHIFT) | jump to first character of line INSERT | go to the first character of line into INSERT |
| A(SHIFT) | jump to last character of line INSERT  |  go to the last character of line into INSERT |
