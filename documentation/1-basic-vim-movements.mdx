# Movement Keys

### single movement


| Shortcut | Alias               | Function            |
| -------- | ------------------- | ------------------- |
| j k      | vertical movement   | move up and down    |
| h l      | horizontal movement | move left and right |


### word movements


| Shortcut  |       Alias       |                            Function |
| :-------- | :---------------: | ----------------------------------: |
| w         |     word hop      |            hop word(exclude spaces) |
| W         |   true word hop   |          hop forward by actual word |
| :-------- | :---------------: | ----------------------------------: |
| b         | backward word hop | hop word backwards (exclude spaces) |
| B         | true backword hop |        hop backwards by actual word |


### editing commands

| Shortcut |       Alias       |                       Function |
| -------- | :---------------: | -----------------------------: |
| y        |       yank        |             yank (buffer save) |
| yy       |    double yank    |      yank a line (buffer save) |
| p        |        put        |                   paste a line |
| -------- | :---------------: | -----------------------------: |
| dd       |   double delete   | delete that line (buffer save) |

\***imp** *yank* and *delete* share the same buffer

| Shortcut | Alias |                        Function |
| -------- | :---: | ------------------------------: |
| u        | undo  | undoes the change (buffer save) |
| CTRL + r | redo  |   redoes the undo (buffer save) |

| -------- | :--------------: | ----------------------------------------------: |
